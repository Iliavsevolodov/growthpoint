# Точка роста

Презентационный сайт пилотного проекта для женщин 40+: развитие коммуникации, личного бренда, контента, навыков искусственного интеллекта и создание собственной команды.

## Структура

- `index.html` — содержание и структура презентации;
- `styles.css` — адаптивная визуальная система проекта;
- `script.js` — анимации появления, слайдер и декоративные микроанимации;
- `.github/workflows/pages.yml` — автоматическая публикация на GitHub Pages.

## Визуальная концепция

Молочная и кремовая основа, солнечно-жёлтые и золотые акценты, тёмно-коричневая типографика, бумажные фактуры, подсолнухи и мягкие природные анимации.

## Публикация

Workflow автоматически публикует содержимое ветки `main` в GitHub Pages после каждого обновления. В настройках репозитория в разделе **Settings → Pages → Build and deployment** источником должен быть выбран **GitHub Actions**.

После активации Pages сайт будет доступен по адресу:

`https://iliavsevolodov.github.io/growthpoint/`
